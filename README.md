# 交互式图表可视化工具

这是一个基于第2章.ipynb文件创建的交互式图表网站，支持多种图表类型的实时数据编辑和显示。

## 功能特点

- **多种图表类型**: 包含折线图、柱状图、堆积柱状图、条形图、堆积条形图、直方图、饼图、圆环图、散点图、气泡图、箱形图、雷达图、误差棒图、面积图等14种图表类型
- **实时数据编辑**: 支持手动修改数据并实时更新图表
- **智能数据解析**: 支持不同图表类型的专用数据格式
- **响应式设计**: 适配不同屏幕尺寸
- **动画效果**: 流畅的图表切换和更新动画

## 使用方法

### 1. 选择图表类型
在"选择图表类型"下拉菜单中选择想要显示的图表类型：
- 折线图 (Line Chart)
- 柱状图 (Bar Chart)
- 散点图 (Scatter Chart)
- 饼图 (Pie Chart)
- 面积图 (Area Chart)

### 2. 编辑数据
在"数据编辑"区域：
- **标签**: 输入数据标签，用逗号分隔（例如：一月,二月,三月）
- **数值**: 输入对应数值，用逗号分隔（例如：10,20,30）
- **图表标题**: 输入图表标题

### 3. 使用示例数据
点击示例数据按钮快速加载预设数据：
- **温度数据**: 城市月平均温度变化
- **销售数据**: 季度销售额统计
- **人口数据**: 人口年龄分布
- **调查数据**: 用户满意度调查结果

### 4. 实时更新
修改数据后，图表会自动更新（500ms防抖）。也可手动点击"更新图表"按钮。

## 技术实现

- **前端框架**: 纯HTML/CSS/JavaScript
- **图表库**: Chart.js
- **响应式设计**: CSS Grid和Flexbox
- **动画效果**: CSS动画和JavaScript动画

## 启动方法

1. 直接在浏览器中打开 `index.html` 文件
2. 或使用本地服务器（推荐）：
   ```bash
   # 使用Python内置服务器
   python -m http.server 8000
   
   # 或使用Node.js http-server
   npx http-server
   ```

3. 在浏览器中访问 `http://localhost:8000`

## 兼容性

- 现代浏览器（Chrome、Firefox、Safari、Edge）
- 移动设备支持
- 需要网络连接加载Chart.js库

## 注意事项

- 确保标签和数值的数量一致
- 数值支持小数和负数
- 数据编辑支持实时更新
- 图表切换时保持数据不变

## 开发说明

基于第2章.ipynb文件中的折线图概念，扩展了更多图表类型和交互功能，提供了比原始文件更丰富的可视化体验。
